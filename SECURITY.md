# Security & Ethics Policy

## 1. Educational Purpose Only
This project is strictly for educational and research purposes in the context of Cyber Security (e.g., M.Sc. Cyber Security studies). It is designed to help professionals understand exploit chains to build better defense and detection systems.

## 2. Responsible Use
- **Isolated Environments:** All simulations MUST be performed in isolated, air-gapped, or strictly controlled virtual environments (e.g., VirtualBox, Proxmox, Docker).
- **No Production Impact:** Never run any exploit simulation against production systems or networks you do not own or have explicit, written permission for.
- **No Redistribution of Weaponized Code:** This lab focuses on the *mechanics* and *detection* of exploits, not on providing ready-to-use tools for malicious intent.

## 3. Responsible Disclosure
If you discover a real-world vulnerability during your research, follow the standard **Responsible Disclosure** process: notify the vendor and give them adequate time to patch before public discussion.
